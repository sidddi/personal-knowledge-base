# Demo Vault

This directory contains a fictional but realistic example of what a populated knowledge base looks like after a few weeks of use.

Open this folder as an Obsidian vault (File → Open Vault → select `demo_vault/`) to explore the structure before building your own.

## What's included

- **3 wiki entries** on AI topics, written as if ingested from real articles
- **1 raw file** ready to be ingested — run `"Ingest raw/2026-05-06_example-rag-in-healthcare.md"` in Claude Code to see the workflow in action
- **Index, hot cache, and log** showing what they look like in practice

## Fictional context

The demo vault simulates a knowledge base for a hypothetical AI architect working on healthcare AI deployments. All organizations, people, and specific figures are fictional.
